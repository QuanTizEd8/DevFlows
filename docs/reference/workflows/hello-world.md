# Hello World

Minimal reusable workflow used to validate DevFlows tooling and CI.

## Usage

```yaml
jobs:
  call-hello-world:
    uses: owner/devflows/.github/workflows/hello-world.yaml@hello-world/v1
```

## Versioning

- Current line: `hello-world/v1`
- Exact release tags use `hello-world/vX.Y.Z`.
- Commit SHAs remain the highest-assurance reference.

## Inputs

| name | type | required | default | description |
| --- | --- | --- | --- | --- |
| message | string | False | Hello from DevFlows | Message to include in the generated greeting. |

## Secrets

None.

## Outputs

| name | description |
| --- | --- |
| greeting | Greeting generated by the workflow. |

## Permissions

- `contents`: `read`
